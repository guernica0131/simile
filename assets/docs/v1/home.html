What is Simile?

Simile is an OPEN web service API for delivering and sharing course content. It uses restful weservices for sending and recieving course data. Developers can build clientside applications (including mobile apps) around simile. Not a developer? We also have ready made clients for delivering your course work.

Some features

  • Course creation
  • Lessons
  • Assessment and Quizzing
  • Assingmnet management
  • Grade managment
  • Portfolio and content sharing
